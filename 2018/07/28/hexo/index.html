注册Disqus
到 http://disqus.com/ 注册 disqus 帐号。

注册 Disqus 帐号，用邮箱注册下就可以了，注册需要填写：Site URL，Site Name，Site Shortname。

提示：一个 disqus 帐号可以申请多个 Site Shortname。不同空间的文章 ID 有可能重合，所以请为不同的空间配置不同 Site Shortname。
